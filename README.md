🚀 Advanced Python Learning Project

This repository covers Advanced Python concepts including Object-Oriented Programming (OOP), Decorators, and Asynchronous Programming (Async / Await).
The goal is to move from intermediate to advanced level by understanding real-world implementation patterns and writing clean, scalable code.




### 1️⃣ Object-Oriented Programming (OOP)

* Classes & Objects
* Constructors (`__init__`)
* Instance & Class Variables
* Methods (Instance, Class, Static)
* Inheritance
* Polymorphism
* Encapsulation
* Abstraction
* Magic (Dunder) Methods

---

### 2️⃣ Decorators

* Function Decorators
* Decorators with Arguments
* Nested Decorators
* Class-based Decorators
* Real-world Use Cases

  * Logging
  * Authentication
  * Execution Time Measurement
  * Caching

---

### 3️⃣ Asynchronous Programming

* Synchronous vs Asynchronous
* `async` and `await`
* Event Loop
* `asyncio` Module
* Creating Coroutines
* Running Concurrent Tasks
* Handling Async API Calls
* Async File Handling

---

## 🎯 Learning Objectives

* Understand advanced Python internals
* Write clean and reusable OOP-based code
* Improve performance using async programming
